Arithmetic operators are used to perform mathematical calculations.

✅ List of Arithmetic Operators

/	Division (gives double)	5 / 2	2.5

main() {
  var x = 20;
  var y = 3;

  print(x + y);
  print(x - y);
  print(x * y);
  print(x / y);
  print(x ~/ y);
  print(x % y);
  print(-(x - y));

  print("x + y = ${x + y}");
  print("x - y = ${x - y}");
  print("x * y = ${x * y}");
  print("x / y = ${x / y}");
  print("x ~/ y = ${x ~/ y}");
  print("x % y = ${x % y}");
  print("-(x - y) = ${-(x - y)}");
}


Unary Operator
Unary operators work with only one operand.

✅ List of Unary Operators
Operator	Name	Example	Result
++	Increment	a++	Increases value by 1
--	Decrement	a--	Decreases value by 1
-	Unary minus	-a	Changes sign
!	Logical NOT	!true	false
~	Bitwise NOT	~5	-6

Prefix Operator
Operator comes before the variable.
++a;
--a;

Postfix Operator
Operator comes after the variable.
a++;
a--;

Bitwise NOT (~)
Bitwise NOT is a unary operator that inverts (flips) every bit of a number.
~a;


main() {
  //prefix postfix
  var a = 5;
  print("a = $a"); //5
  print("pre increment ${a++}"); //5
  print("post increment ${++a}"); //7
  print("final a = $a"); //7
  a = 5;
  print("a = $a"); //5
  print("pre decrement ${a--}"); //5
  print("post decrement ${--a}"); //3
  print("final a = $a"); //3

  //Logical NOT
  var istrue = false;
  print(!istrue);

  //Unary Minus
  var b = 10;
  print("b = $b");
  print("Unary minus of b = ${-b}");

  //Bitwise NOT
  var c = 5;
  print("c = $c");
  print("Bitwise NOT of c = ${~c}");
}

output:
a = 5
pre increment 5
post increment 7
final a = 7
a = 5
pre decrement 5
post decrement 3
final a = 3
true
b = 10
Unary minus of b = -10
c = 5
Bitwise NOT of c = -6

Relational Operators

Relational operators are used to compare two values.
They always return a boolean value: true or false.

✅ List of Relational Operators
Operator	Meaning	Example	Result
==	Equal to	5 == 5	true
!=	Not equal to	5 != 3	true
>	Greater than	7 > 3	true
<	Less than	3 < 7	true
>=	Greater than or equal to	5 >= 5	true
<=	Less than or equal to	4 <= 6	true


main() {
  var a = 20;
  var b = 10;
  print("a = $a");
  print("b = $b");
  print("Is a and b equal? : ${a == b}");
  print("Is a and b not equal? : ${a != b}");
  print("Is a greater than b? : ${a > b}");
  print("Is a less than b? : ${a < b}");
  print("Is a greater than or equal to b? : ${a >= b}");
  print("Is a less than or equal to b? : ${a <= b}");
}

output:
a = 20
b = 10
Is a and b equal? : false
Is a and b not equal? : true
Is a greater than b? : true
Is a less than b? : false
Is a greater than or equal to b? : true

Bitwise Operators

Bitwise operators are used to manipulate individual bits of integer values.

✅ List of Bitwise Operators
Operator	Name	Meaning	Example	Result
&	AND	Bits are 1 if both are 1	5 & 3	1
|	OR	Bits are 1 if any is 1	`5
^	XOR	Bits are 1 if bits are different	5 ^ 3	6
~	NOT	Inverts all bits	~5	-6
<<	Left shift	Shifts bits left	5 << 1	10
>>	Right shift	Shifts bits right	5 >> 1	2

main() {
  var x = 15; //01111
  var y = 11; //01011

  print(x & y); //01011
  print(x | y); //01111
  print(x ^ y); //00100
  print(~x);
  print(~y);
  print(x >> 1);
  print(x << 1);
  print(y >> 1);
  print(y << 1);
}

Dart Encapsulation
Encapsulation means hiding internal data (variables) and controlling access to it through methods (getters/setters).
Protect data from misuse
Control how data is modified
Improve security
Make code maintainable

class Encapsulationclass {
  String _firstName; // _firstName is private variable
  String _lastName; // _lastName is private variable
  int _age; // _age is private variable
  String _gender; // _gender is private variable

  Encapsulationclass(this._firstName, this._lastName, this._age, this._gender) {
    print("Birth Year: ${_getBirthYear()}");
  }
  //getter method
  String getFullName() {
    return _firstName + " " + _lastName;
  }

  //setter method
  void changeGender(String gender) {
    if (gender == 'Male' || gender == 'Female') {
      _gender = gender;
    }
  }

  String getGender() {
    return _gender;
  }

  void setAge(int newAge) {
    if (newAge > 18 && newAge < 70) {
      _age = newAge;
    }
  }

  int getAge() {
    return _age;
  }

  int _getBirthYear() {
    return 2026 - _age;
  }
}

import 'EncapsulationClass.dart';

main() {
  Encapsulationclass Person = Encapsulationclass(
    'Priyanto',
    'Debnath',
    22,
    'Male',
  );
  print(Person.getFullName());

  Person.changeGender('male');
  print(Person.getGender());

  Person.setAge(23);
  print(Person.getAge());
}

Birth Year: 2004
Priyanto Debnath
Male
